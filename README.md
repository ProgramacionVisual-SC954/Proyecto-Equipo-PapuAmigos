FixHome - Sistema de Gestión de Empleados

Proyecto desarrollado para la materia Programación Visual (SC954) utilizando Visual Basic .NET 8 y Windows Forms.

Descripción

Sistema para la gestión de empleados de FixHome, una plataforma que conecta usuarios con técnicos para servicios de reparación y mantenimiento del hogar.

La aplicación permite:

* Registrar empleados.
* Consultar empleados.
* Editar información.
* Eliminar empleados.
* Calcular pagos mensuales.
* Mostrar el total de empleados y nómina.

Tecnologías

* Visual Basic .NET 8
* Windows Forms
* Git y GitHub
